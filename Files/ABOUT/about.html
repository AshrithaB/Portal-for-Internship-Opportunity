The project titled "Portal For Internship Opportunities" is internship software for monitoring and accessing internships based on the user interests for taking the internships from their preferred companies . The main objective of the "Internship Management System" is to give students an online platform for more organised and professional technique to regulate such opportunities so that all students are intimated whenver a new opportunity comes up.
The purpose of the project is to build an application program to reduce the manual work for students who find it hard to come across any suitable internship. This project is developed using HTML, CSS and JavaScript for front-end and PHP, MySQL for back-end .
